Добро пожаловать в Exordos Core!

**Exordos** — это open-source NoOps платформа для управления инфраструктурой на всех уровнях, а также экосистема, построенная на её основе.

## 📦 Установка

### Linux

```bash
curl -fsSL https://repository.genesis-core.tech/install.sh | sudo sh
```

## 🚀 Руководство для разработчиков приложений

Всё, что нужно для разработки и развёртывания приложений на платформе Exordos Core. В терминологии платформы приложения называются **элементами**. Руководство охватывает все этапы жизненного цикла элемента:

- **Написание манифеста** — определение структуры элемента, его зависимостей и конфигурации.
- **Сборка** — упаковка элемента и его артефактов.
- **Публикация** — выпуск элемента в реестр экосистемы.
- **Развёртывание** — установка и запуск элемента на платформе.

[Перейти к руководству →](app-developer-guide/index.ru.md)
