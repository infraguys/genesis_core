# Примеры

## Сборка элемента из репозитория

Сборка элемента Exordos. Команда собирает все образы, манифесты и другие артефакты, необходимые для элемента. Манифест в проекте может быть raw YAML файлом или шаблоном, использующим Jinja2. Для шаблонов Jinja2 по умолчанию доступны следующие переменные:

- {{ version }}: версия элемента
- {{ name }}: имя элемента
- {{ images }}: список образов
- {{ manifests }}: список манифестов

Перейдите в директорию проекта и выполните следующую команду:

```bash
exordos build .
```

## Установка элемента из репозитория

```bash
exordos elements install <element_name>
```
